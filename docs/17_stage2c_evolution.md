## 一、扩池：从 72 只到 454 只（6.3×）

**问题**：M17 的 `US_POOL`（~80 只大盘）让价值/成长信号被池子构成绑架——
小池 mega-cap 偏价值，全池偏成长，`select_stable_positive` 在两种构成下
筛出不同的因子。

**动作**：构建 `EXPANDED_US_POOL`（S&P 500 + S&P 400 + 精选小盘，513 只），
通过 `scripts/expand_us_pool.py` 批量拉取行情+基本面数据缓存。

**行业映射**：因 akshare 的 `stock_individual_basic_info_us_xq` 接口失效，
手工构建 GICS 一级行业映射（`industry_us.py`），覆盖全部 513 只、11 个行业。

**结果**：
- 行情 100% 可用（513/513），基本面 87% 可用（454/513）
- 金融股（JPM/BAC/GS 等 9 只）和 REITs（SPG/PLD/WELL 等 28 只）基本面返回 None
- 基准收益从 M17 的 +344%（72 只大盘）暴跌到 +68%（含中小盘）——中小盘在 2018-2025
  大幅跑输大盘。但策略仍有 +639% 收益，超额从 +143pct 飙到 +571pct

## 二、行业中性化：剥 β 留 α

**逻辑**（与 A 股 M12 一致）：因子分数里的行业成分是 β 不是 α
（如价值因子天然偏银行/能源）。行业中性化剥掉行业均值后，剩下的是
行业内相对排序——更接近「这家公司比同行好多少」。

**实现**：`neutralize(factor, industry=ind, mode="industry")`，
单日截面内完成（无未来函数）。

**结果**：
```
原始 3 因子 → 行业中性化：
  quality_roe: ICIR +0.07/+0.07 → +0.03(t+1.44)，变弱但仍可用
  quality_gm:  ICIR +0.07 → +0.11(t+4.91)，明显提升
  growth_rev:  ICIR +0.06 → +0.13(t+5.99)，明显提升

组合层面：
  等权(原始):  67% 跑赢 | 超额夏普 +0.21
  等权(中性化): 75% 跑赢 | 超额夏普 +0.36
```

中性化减少了稳定因子数（3→2），但剩余因子纯度更高、组合更强。

## 三、多空对冲：诚实负结果（跨市场一致）

**逻辑**：美股可做空，L5 多头+L1 空头理应对冲掉市场 β 和方向漂移，
提取纯因子 α。

**结果**（全池 454 只，含借券费 2%/年）：
```
L1→L5 单调性：全 7 个因子均不单调（L1 股票本身不差）
多空跑赢率：  25%（vs 多头 75%）
多空全样本：  +105% 收益 / 夏普 0.36（vs 多头 +328% / 夏普 0.90）
```

**根因**：分层最底部的股票不是「烂公司」——它们只是因子分数最低。
在成长牛市中，高估值/低质量/低成长的「因子底部」往往是跑赢的成长股。
做空 L1 = 做空成长股 = 亏钱。

这与 A 股 M12「L5−L1 夏普≈0」的发现**跨市场完全一致**。
因子 α 在「选最好的」方向有效，「做空最差的」方向无效。

## 四、量价因子：正交信号的突破

**问题**：M16 在 80 只大盘池中，`select_stable_positive` 筛出 0 个量价因子。
但那是「方向不净正向」——多数量价因子 IC 为负是因为成长牛市中
高波动/低流动性股票跑赢。**方向稳定≠方向为正**。

**方法**：改用 **IC 方向稳定性筛选**（|t|>2，不要求 posRate 高）。
方向明确为负的因子翻转后使用。

**结果**（扩展 454 只池）：
```
6 个量价因子中 5 个通过方向筛选：
  pv_momentum60: IC-0.007(t-2.17) → 翻转 ✅
  pv_reversal20: IC+0.009(t+3.39) → 正向 ✅
  pv_lowvol20:   IC-0.013(t-3.98) → 翻转 ✅
  pv_amihud:     IC-0.012(t-4.09) → 翻转 ✅
  pv_parkinson:  IC-0.012(t-3.30) → 翻转 ✅
  pv_maslope20:  t-1.91 → 方向不稳 ❌

量价 vs 基本面交叉相关：平均 0.038（近乎完美正交！）
```

**组合效果**：
```
纯基本面 3F:     +639%  夏普 0.95  回撤 36.1%
基本面+量价 8F:  +1003% 夏普 0.98  回撤 28.4%  ← 收益+57%, 回撤-21%
```

## 五、多周期因子：进一步扩容

**方法**：量价因子做 5d/20d/60d 多周期版本；基本面做变动因子
（毛利率 YoY 变化、营收增长率加速度）。

**新增 7 因子全部通过方向筛选**，最强新因子是 `gm_yoy_change`
（毛利率同比变化，t=+7.02）。

**最终 15 因子组合**：
```
+1278% 累计 / 夏普 1.04 / 回撤 30.7% / 滚动跑赢 67% / 超额夏普 +0.14
```
