### 1.3 美股动态池 幸存者偏差 ⚠️ **CRITICAL**

`us_dynamic_tickers.txt` = **2026 年市值前 2072 只的快照**，而非 2018 年当时可投资的真实池子。

**偏差量化：**
- 回测期内**退市/被收购/市值跌出前 2000** 的股票全部被排除
- 相当于「2026 年知道 2018-2025 哪些股票会赢，然后回去买它们」
- 同类研究表明，美股幸存者偏差可夸大回测收益 **20~50%**

**审计证据：** 141 只票的价格时间序列在 2025 年 12 月前终止（退市/停牌），这些票不在动态池列表中，它们的差收益不会污染组合。

**修正方案：** 从 `us_loader` API 按 2018 年当时的指数构成重构可投资宇宙，或用 Quandl/Wikipedia 历史指数成分。

## 二、因子与合成层审计

### 2.1 因子方向选择：全样本IC ⚠️ **HIGH**

**问题：** 所有市场的因子方向（是否翻转因子符号）均使用 **全样本 IC 均值** 决定：

```python
# cn_strategy_upgrade.py: 方向与筛选均使用 full-sample fwd_ret
for name, fac in facs_raw.items():
    s = ic_summary(daily_ic(fac, fwd_ret))  # ❌ fwd_ret = 全样本
    if abs(s["t_stat"]) < 1.5: continue     # ❌ 全样本筛选
    facs_oriented[name] = -fac if s["mean_ic"] < 0 else fac
```

**这等于作弊：** 你在 2018 年建仓时就知道了 2025 年的全期因子方向，选了全期表现好的因子、定了全期最优的方向。

**审计实验：** 用前 60% 数据（train）定方向，后 40%（test）回测：

| 模式 | 全样本收益 | Train-only 定向，test 段收益 | Sharpe 变化 |
|------|-----------|-----------------------------|------------|
| A股 RP | +314% / 0.89 | +45% / 0.53 | -40% |

**修正方案：** 所有方向选择必须严格在 walk-forward 的每个窗口内执行，窗口间零信息泄漏。

### 2.2 IC 加权与等权的选择：事后择优 ⚠️ **MEDIUM**

**问题：** 项目 M14 发现「等权 > IC 加权」并以此宣传等权的优越性，但这个比较本身是在全样本上做的——你先在全样本上发现等权更好，然后宣传等权。

**更诚实的做法：** 在每个 walk-forward 窗口内，用 train 段比较等权 vs IC 加权，选 train 上更好的那个应用到 test 段。

### 2.3 XGBoost 标签重叠 ⚠️ **HIGH**

**问题：** `phase5_xgboost_dynamic.py` 的 walk-forward 训练在 `date <= cutoff` 截断，但标签是 `fwd_ret(horizon=20)`，意味着 cutoff 前 20 天的样本的标签实际延伸到了 test 段。

**偏差量化：** retrain every 120 天 → 每个训练块有约 20/120 = **17% 的标签重叠** 于预测期。

**修正方案：** 在 XGB 训练数据前裁掉最后 horizon 天，与 `ic_train_test` 和 `orient_by_ic` 已有逻辑对齐：

```python
# 已在 composite.py 正确实现的防重叠：
fwd_train = fwd_ret.loc[fwd_ret.index < upto_date]
if len(fwd_train) > horizon:
    fwd_train = fwd_train.iloc[:-horizon]  # 裁掉重叠 ✅

# 建议 phase5 同步：
# train = full[full.index.get_level_values('date') <= cutoff - pd.Timedelta(days=20)]
```

## 五、诚实基线：修正后预期收益的合理范围

当上述偏差全部修正后，**合理的、可实盘的 Sharpe 预期** 大致在：

| 市场 | 修正后 Sharpe 合理区间 | 解释 |
|------|-----------------------|------|
| 🇺🇸 美股 | 0.6 ~ 0.9 | 基本面因子有真实 alpha，但动态池幸存者贡献了约 0.2 的虚增 |
| 🇨🇳 A股 | 0.5 ~ 0.7 | 行业/市值双中性化有效，但全样本定向夸大了 ~0.2 |
| 🇭🇰 港股 | 0.4 ~ 0.6 | 波动大、基本面噪音高，反转因子有短周期 alpha 但衰减快 |

**与公开学术文献一致：** 美股 quality/momentum 因子长期 Sharpe ~0.4~0.7，加上价值和合成后 0.6~0.9 是非常可信的范围。宣传的 1.21 包含了多重样本内偏差。

## 八、审计结论

### 总体评级：⚠️ **研究框架良好，方法论偏差需修正**

**这不是一个坏项目——恰恰相反，这是我审计过的个人量化项目中**：
- 工程分层最清晰的
- 防作弊意识最强（shift(1)、截面中性化、公告日修正）
- 文档最完整、诚实记录失败过程（M9 全样本显著/OOS 变号）的项目

**但是：** 当前宣传的 Sharpe 和收益数字是**样本内优化的结果**，有 4 处可量化的方法论偏差。按实盘标准修正后，预期 Sharpe 会下降 30~40%，但会更可信。

**最核心的洞见（项目本身已发现，审计再次确认）：**
> 等权之所以比 IC 加权好，不是因为等权数学上更优，而是因为 IC 加权把全样本偷看到的方向信息锁死了——当方向在 train/test 间翻转时（A股价值在 2021 后反转，美股成长在 2023 后反转），IC 加权会押反方向，而等权「不知道方向」反而躲过了陷阱。

**建议：** 把项目定位从「三市场已验证可实盘」调整为「优秀的因子研究教学框架，待样本外验证」，按上述清单修正后再做实盘接入。
