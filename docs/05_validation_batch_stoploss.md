# 05 · 样本内外切分、多标的批量回测、止损

> 接 `04`。前面所有回测都犯了同一个隐患：**用全部历史数据**。这一章修正它（B 样本内外切分），把策略推广到**一篮子股票**（C 批量回测），并给均值回归加上**止损**这个可选保险（D）。

---

## 一、B：样本内 / 样本外切分（检验过拟合的金标准）

**问题**：我们在 03 章用全部数据找「最优参数」。但「在已知答案上考满分」毫无意义——真正要回答的是「这组参数换到**没见过的数据**上还灵吗？」

**做法**：把时间序列切成两段（注意：时序数据**不能随机打乱**，只能按时间先后切）：
- **样本内 / 训练集 (in-sample / train)**：用它来**挑最优参数**。
- **样本外 / 测试集 (out-of-sample / test)**：用上一步选出的参数，在这段**全新数据**上跑，看表现。

**怎么判读**：
- 训练集好、测试集也不错 → 参数抓住了真实规律，**可信**。
- 训练集惊艳、测试集垮掉 → 典型**过拟合**，那个「最优」是拟合噪声拟合出来的。

> 这就是机器学习里 train/test 的思想搬到回测上。它是「玩具回测」和「可信研究」的分水岭。
> 更严格的版本叫 **walk-forward**（滚动多次切分），我们先做最基础的一次切分建立直觉。

---

## 二、C：多标的批量回测（避免「一只股票的运气」）

**问题**：前面只在 1~2 只股票上验证。万一均值回归只是「恰好」适合平安银行呢？单只股票的结论可能是运气。

**做法**：在**一篮子股票**（如沪深主要权重股）上跑**同一个策略、同一组参数**，汇总每只的绩效，再看**横截面统计**：
- 胜率：多少比例的股票跑赢了买入持有？
- 收益/夏普的**中位数**（比均值更抗极端值）。
- 分布是否稳定，还是靠个别股票拉高。

**怎么判读**：一个策略只有在**大多数标的上都站得住**，才值得相信它抓住了普适规律，而非个例。

---

## 三、D：止损（作为策略的可选参数）

**问题**（04 章埋的雷）：均值回归会「接飞刀」——标的单边阴跌时，z 持续为负，策略越跌越买、一路被套。

**止损**：给持仓设一条「亏损红线」。**从买入后的成本价算起，浮亏超过 X%（如 8%）就无条件清仓**，哪怕策略信号还说「持有」。它牺牲了一部分「也许会反弹」的可能，换取「不被单边趋势打穿」的保护。

**工程设计（呼应你的提议）**：把止损做成**可选参数** `stop_loss`：
- 默认 `None` → 关闭，完全不影响原有行为（向后兼容）。
- 传入如 `0.08` → 开启 8% 止损。
- 这样同一个策略函数，既能跑「无止损」也能跑「有止损」，**便于 A/B 对比止损到底有没有用**。

> 止损会改变信号，所以仍在策略层实现，并同样 `shift(1)` 防未来函数。
> 注意止损不是「越紧越好」——太紧会被正常波动频繁触发（在低点割肉），太松则形同虚设。这本身又是一个要权衡的参数。
